Get started

    API Endpoint

    POST /api/analyze-spending

The Spending Analysis analyze your spending patterns and receive recommendations for better financial management.

This API allows users to submit transaction data in JSON format and provides insights along with recommendations.

get characters


//To send transactional data as JSON,
//you can use this example with fetch in JavaScript:
fetch('http://127.0.0.1:5000/api/analyze-spending', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      // Replace with your actual transactional data
      "transactions": [
        {
            "date": "2024-01-15",
            "category_name": "Groceries",
            "amount": 150,
            "transaction_type": "expense"
        },
        {
            "date": "2024-01-20",
            "category_name": "Entertainment",
            "amount": 80,
            "transaction_type": "expense"
        },
        {
            "date": "2024-01-30",
            "category_name": "Rent",
            "amount": 1000,
            "transaction_type": "expense"
        }
      ]
    })
  })
  .then(response => response.json())
  .then(data => {
    console.log('Success:', data);
  })
  .catch((error) => {
    console.error('Error:', error);
  });

To get characters you need to make a POST call while submmiting JSON Transactional data to the following url :

JSON Explaination

Field Type Description
spending_analysis Array Array of analyzed transactions.
date date Date of the transaction.
category_name String Name of the transaction category.
amount Float Actual transaction amount.
predicted_amount Float Predicted transaction amount based on analysis.
recommendations Array Array of recommendations based on analysis.
